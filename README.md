# Telegram Bot для продажи вебинаров

Этот бот создан для продажи вебинаров Анны Тельновой по нумерологии с использованием Telegram Bot API и Flask для webhook.

## Функциональность

Бот реализует следующий сценарий:

1. **Начало** (`/start`) - Приветственное сообщение с описанием предложения
2. **"Хочу вебинар!"** - Выбор конкретного вебинара для покупки
3. **"Все еще сомневаюсь"** - Дополнительная информация для сомневающихся
4. **"Остались сомнения"** - Предложение бесплатной половины вебинара
5. **Выбор вебинара** - Отправка ссылки на покупку или бесплатную половину

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Отредактируйте файл `config.py`:

```python
# Токен бота от @BotFather
BOT_TOKEN = "ваш_токен_бота"

# URL для webhook (замените на ваш реальный IP адрес)
WEBHOOK_URL = "https://ваш_ipv4_адрес/bot"

# Список вебинаров
WEBINARS = [
    "Энергопоток",
    "Денежный рост", 
    "Пакет сверхмощных практик"
]

# Ссылки для покупки
PURCHASE_LINK = "https://ваш_сайт.com/purchase"
HALF_WEBINAR_LINK_TEMPLATE = "https://ваш_сайт.com/{webinar}-half"
```

### 3. Получение токена бота

1. Найдите @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен в `config.py`

### 4. Настройка webhook

Для работы webhook ваш сервер должен быть доступен из интернета. У вас есть несколько вариантов:

#### Вариант A: Использование ngrok (для тестирования)

```bash
# Установите ngrok
# Скачайте с https://ngrok.com/

# Запустите туннель
ngrok http 80

# Скопируйте полученный URL и обновите WEBHOOK_URL в config.py
# Например: https://abc123.ngrok.io/bot
```

#### Вариант B: VPS сервер

1. Разверните сервер с публичным IP
2. Настройте домен (опционально)
3. Обновите WEBHOOK_URL в config.py

## Запуск

### Запуск бота

```bash
python main.py
```

### Проверка работы

1. Откройте браузер и перейдите по адресу `http://ваш_ip:80/`
2. Должно появиться сообщение "Telegram Bot is running!"
3. Найдите вашего бота в Telegram и отправьте `/start`

## Настройка вебинаров

Для изменения списка вебинаров отредактируйте массив `WEBINARS` в файле `config.py`:

```python
WEBINARS = [
    "Название вебинара 1",
    "Название вебинара 2",
    "Название вебинара 3"
]
```

## Настройка ссылок

Обновите ссылки в `config.py`:

```python
# Ссылка для покупки полного вебинара
PURCHASE_LINK = "https://ваш_сайт.com/purchase"

# Шаблон для ссылок на половину вебинара
HALF_WEBINAR_LINK_TEMPLATE = "https://ваш_сайт.com/{webinar}-half"
```

## Безопасность

- Никогда не публикуйте токен бота в открытом доступе
- Используйте HTTPS для webhook в продакшене
- Регулярно обновляйте зависимости

## Логирование

Бот выводит базовую информацию в консоль. Для продакшена рекомендуется настроить полноценное логирование.

## Поддержка

При возникновении проблем:

1. Проверьте правильность токена бота
2. Убедитесь, что webhook URL доступен из интернета
3. Проверьте логи сервера
4. Убедитесь, что все зависимости установлены
